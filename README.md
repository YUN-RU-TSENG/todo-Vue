## 項目簡介

<div align="center">
    <img src="https://i.imgur.com/yH8ADnF.png" alt="project img" width="80%"/>
</div>

簡易的 TODO List，可以隨手紀錄代辦事項。
設計稿為自行設計。

## 項目功能

* CRUD 代辦事項
* 區分完成、未完成待辦事項
* 可針對代辦事項添加日期、重要程度
* 表單驗證功能（採用 HTML5 表單驗證）

## 項目技術

### Vanilla JS
* Javascript(Proxy)
* SCSS、HTML5
* Dayjs

### Vue2
* CDN Vue2
* SCSS、HTML5
* Dayjs

### Vue3
* Vue CLI
* Vue3
* SCSS、HTML5
* Dayjs

### Installation

1. Clone the repo
   ```sh
   https://github.com/YUN-RU-TSENG/todo-list
   ```
2. Install NPM packages (Vue3 ver)
   ```sh
   cd vue3
   npm install
   ```
